# SH-Sharp
PHP+C# for the command line. Yes, that's right.

```c#

<#zsh
    echo "How could this be useful?"
#>

for (var i = 0; i < 10; i++) {
<#zsh
    echo "I'll give you {i} reasons"
#>
}

<#zsh
files=$(ls *)
#>

foreach (var file in Var["files"].Split('\n')) {
    Console.WriteLine(file);
}

```
